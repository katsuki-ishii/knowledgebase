# Knowledgebase

個人用の学習・メモをまとめた知識ベース。  
AWS / CS / JS / Vue.js / dev tools などのトピックを、短いメモとして整理している。  
日々の学習ログを積み上げることを目的にしている。

## 収録内容
- AWS: 認証/権限まわりや運用メモ
- CS: 基本概念・プロトコル・仕様メモ
- JS: 基礎概念や言語仕様の理解メモ
- Vue.js: 基礎からコンポーネント設計までの学習メモ
- dev tools: ESLint / Prettier など開発ツール関連メモ

## ディレクトリ構成
- `Knowledgebase/` 以下にカテゴリ別で配置
- 各ファイルは短い要点メモ形式

## 使い方
- Obsidian で閲覧する想定（`.obsidian` の設定を含む）
- 検索・タグ・リンクでナレッジを横断的に参照できる

## 方針
- 分量よりも「理解の要点」を優先する
- 追記・修正で継続的に更新する
