# [[ARPANET]]まとめ

## 1. [[ARPANET]]とは何か

[[ARPANET]]（[[ARPANET|Advanced Research Projects Agency Network]]）は、1969年にアメリカ国防総省の研究機関ARPAが開発した世界初の実用的なパケット通信ネットワークである。

現在のインターネットの直接の祖先にあたる。

---

## 2. なぜ作られたのか

背景は冷戦期である。

目的は主に次の三つである。

1. 高価なコンピュータ資源を研究機関間で共有するため
2. 研究を効率化し、技術的優位を確保するため
3. 障害に強い分散型通信の研究を行うため

「核戦争用ネットワーク」と言われることがあるが、正確には分散型ネットワーク研究の一環である。

---

## 3. どこから始まったか

1969年、最初に接続された4拠点は以下である。

* UCLA
* スタンフォード研究所（SRI）
* UCサンタバーバラ
* ユタ大学

つまり、実際の運用主体は大学や研究機関であった。

---

## 4. 何が革新的だったのか

最大の特徴は「パケット交換方式」である。

### 従来の電話方式（回線交換）

```
A ─────── B
（通信中は回線を占有）
```

通信が始まると、専用の回線を固定的に占有する方式である。途中で回線が切れると通信は終了する。

### [[ARPANET]]の方式（パケット交換）

```
      ┌─── C ───┐
A ──┬─┤         ├─┬── B
    │  └─── D ───┘  │
    └───────────────┘

データは小さな単位（パケット）に分割され、
複数経路を柔軟に通って目的地へ到達する。
```

経路の一部が故障しても、別ルートを通ることができる。
これが現在のインターネットの基本思想である。

---

## 5. TCP/IPへの移行

1983年1月1日、[[ARPANET]]は通信プロトコルをTCP/IPへ全面移行した。

```
[ネットワークA]
       │
       │
[ネットワークB]──TCP/IP──[ネットワークC]
       │
[ネットワークD]
```

これにより、異なるネットワーク同士を相互接続できるようになった。

この「ネットワークの集合体」がインターネットである。

---

## 6. [[ARPANET]]とインターネットの違い

| [[ARPANET]] | インターネット     |
| ----------- | ----------- |
| 単一の実験ネットワーク | 複数ネットワークの集合 |
| 軍・研究中心      | 世界規模・商用利用   |
| 実験段階        | 社会基盤        |

[[ARPANET]]は1990年に役目を終えて停止したが、その技術思想は現在も引き継がれている。

---

## 7. 具体例で理解する

例：UCLAの研究者がユタ大学のコンピュータを利用したい場合

従来：

* データを物理媒体で郵送する
* 直接現地へ行く

[[ARPANET]]導入後：

* 遠隔ログインが可能
* ファイル共有が可能
* メール送信が可能

研究効率は飛躍的に向上した。

---

## 8. 本質

[[ARPANET]]の本質は単なるネットワークではない。

* 分散
* 相互接続
* オープンな技術共有

この思想が現在のインターネットを生み出した。

[[ARPANET]]はインターネットの原型であり、技術的・思想的出発点である。
